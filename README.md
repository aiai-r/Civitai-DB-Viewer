# 概要

このツールは、Civitai のモデル一覧を見たり、必要なモデルを DB に保存したり、保存済みモデルをあとから見直したりするための Flask アプリです。

# インストールの準備

- Python が必要です
- このプロジェクトは Python 3.11 想定です
- Python 3.10 以上を推奨します
- ダウンロード先: https://www.python.org/downloads/windows/

セットアップ手順

1. Python をインストールします
2. install.batを実行します

# 起動方法

1. `run_flask.bat` を実行します
2. 自動で `http://127.0.0.1:5000/models` を開きます

# 最初にやること

1. 上部メニューの「設定」を開きます
2. Civitai の API キーを入力して保存します
3. 必要に応じて他の設定も行ってください。

- API キーの取得方法: https://note.com/aiaicreate/n/n16d48cfb5758

# メニューの説明

## Civitai

API からモデル一覧を取得して、その場で表示する画面です。

上部フォームで指定できるもの

- ベースモデル
- モデルタイプ
- Period
- Sort
- 取得方式
- 検索モード
- 取得キーワード
- 開始ページ
- 1ページ件数
- 待機(秒)

基本的な使い方

1. 条件を入れます
2. 「表示」を押します
3. 一覧を見て、必要な行だけ保存または DL します

保存とDLの使い分け

- 各行の `DBに保存` / `モデルDL` は、その行だけ個別に実行します
- テーブル下の `選択保存` / `選択DL` は、チェックを入れた複数行をまとめて実行します

検索モードの違い

- Tag: 通常のタグ検索です
- Name: 名前検索です。Civitai API の仕様上、ページ番号ではなく nextPage(cursor) ベースで進みます

## DB

保存済みモデルの一覧を見る画面です。

できること

- 保存済みモデルの確認
- ベースモデルで絞り込み
- モデルタイプで絞り込み
- 名前またはモデル ID で検索
- 複合ソート
- 行ごとの更新

## DB一括取得

Civitai と同じ条件フォームで、一覧表示せずにまとめて DB へ保存する画面です。

基本的な使い方

1. 条件を入れます
2. 「DBへ保存」を押します
3. ログを見ながら完了を待ちます

## 設定

このアプリ全体で使う設定を管理します。

設定できる項目

- APIキー
- NSFW
- 動画を含める
- キャッシュ使用
- キャッシュ保存先
- Base models 手動追加
- Types 手動追加
- preview 保存
- civitai.info 保存
- モデル保存先

補足

- キャッシュは「一覧情報キャッシュ」のみです
- メディアキャッシュ機能はありません
- NSFW は Civitai と DB一括取得の条件に共通です

## Base models / Types の手動追加

Civitai に新しい Base model や Type が追加された場合、通常は「モデルリスト更新」で候補をある程度取り込めるようにしています。

ただし、それでも候補が正しく取得できない場合があります。
そのときは設定画面の `Base models 手動追加` または `Types 手動追加` に直接追加してください。

ここに追加した値は、一覧条件や取得条件の候補として使われます。

注意点

- Civitai 側の表記と完全に合わせて入力してください
- 文字の違い、大文字小文字、空白違いがあると正しく一致しない場合があります
- 取得したいモデルの Civitai 上の表記をそのまま使うのが安全です

# ソートについて

一覧の列ヘッダを押すと並び替えできます。

挙動

- 1回目: 昇順
- 2回目: 降順
- 3回目: 解除

複数列ソート

- 別の列を続けて押すと、複数列の組み合わせでソートできます
- 解除されるまでは、前の列の条件も保持されます
